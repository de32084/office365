Aplicaciones que se incluyen

Veamos las aplicaciones en detalle

Word

Word es uno de los programas de procesamiento y edición de texto más populares en la actualidad. Es utilizado por estudiantes, empresas y el gobierno. También es el programa elegido por muchos autores en el mundo de hoy. Está disponible en versiones para tabletas, así como en versiones en línea. Word se usa a menudo para crear páginas en libros y otros documentos. También se utiliza para crear hojas de vida.

Excel

Excel es una de las aplicaciones de Microsoft Office más poderosas y ampliamente utilizadas en el mundo. Excel es beneficioso para personas y empresas que necesitan administrar números a gran escala, a través de números ajustados de diferentes maneras. También es bueno para manipular y analizar datos, y puede crear gráficos y tablas de aspecto profesional sin mucho esfuerzo. El programa le permite trabajar con múltiples tipos de hojas de cálculo simultáneamente, creando una amplia gama de tablas, gráficos y modelos que pueden usarse para ayudarlo a comprender mejor sus datos.

PowerPoint
Microsoft PowerPoint es un programa de diseño de presentaciones. Le permite crear presentaciones de diapositivas que se ejecutan con sonido, video y animación. También puede crear presentaciones de diapositivas y películas. Tiene la capacidad de hacer hermosas presentaciones y es una herramienta esencial para crear presentaciones y una herramienta útil para los negocios.

One Note
Microsoft OneNote es más que una aplicación para tomar notas. Es una herramienta organizativa que te permite recopilar y compartir toda la información, ideas y tareas que te interesan. OneNote es una aplicación completa por sí sola, disponible para Mac, PC, tabletas y teléfonos con Windows, y puede usarla para planificar, intercambiar ideas, organizar, compartir y colaborar en proyectos, reuniones y otras actividades.

Publisher (solo PC)
Microsoft Publisher es un software de publicación de escritorio que le permite crear varias cosas, como carteles, volantes y tarjetas de felicitación. Es parte de Microsoft Office y se utiliza para crear documentos de calidad profesional, incluidos boletines, folletos, carteles, volantes, tarjetas de felicitación, etc. Incluso puede usarlo para crear PDF interactivos y boletines por correo electrónico. Las herramientas intuitivas de diseño y diseño de Publisher lo ayudan a contar su historia de una manera que es fácil de compartir e imprimir.

Access (solo PC)
Microsoft Access es una aplicación de base de datos que forma parte del conjunto de productos de software de Microsoft Office. Es una herramienta completa y poderosa que permite a los usuarios crear bases de datos personalizadas para rastrear todo tipo de información, ya sea información financiera, registros de clientes, libro de cuentas, etc., direcciones personales o una lista de programas de TV favoritos. No requiere ningún servidor para su funcionamiento. Algunas de sus funciones son la importación y exportación, el filtrado, la actualización automática, la exportación, la impresión, etc.

Outlook
Microsoft Outlook es un programa de programación y correo electrónico que organiza y almacena los correos electrónicos, las citas, los contactos y otra información personal de los usuarios. También se puede utilizar para gestionar tareas y reuniones y agilizar la comunicación entre compañeros. Es un poderoso sistema de administración de correo electrónico, calendario y tareas,

SharePoint
El servidor de SharePoint es una herramienta utilizada para almacenar, organizar y compartir documentos e información. Puede ayudarlo a trabajar de manera más eficiente al agilizar sus tareas y facilitar el compartir informes y presentaciones con otras personas en su organización. Pero con tantas funciones y capacidades, no es el producto más fácil de aprender, especialmente si tiene poca o ninguna experiencia con él.

Yammer
Yammer es una aplicación de red social empresarial basada en la nube. Permite a los empleados crear grupos en torno a un tema o proyecto y comunicarse entre ellos. Los usuarios también pueden acceder a los datos de la aplicación a través de las API e integrarlos en sus propios sistemas.

OneDrive
Diseñado para consumidores y estudiantes, Microsoft OneDrive es una solución de almacenamiento en la nube que le permite almacenar, sincronizar y compartir archivos entre sus dispositivos. OneDrive viene con 15 GB de almacenamiento gratuito (que es suficiente para almacenar unos pocos cientos de fotos, un puñado de documentos y algunos videos), y puede ganar hasta cinco gigabytes adicionales gratis al recomendar amigos al servicio. Si bien eso puede no parecer una tonelada de espacio de almacenamiento, OneDrive es ideal para pequeñas empresas con un presupuesto limitado que no quieren gastar dinero extra en costosas suscripciones de almacenamiento en la nube porque ofrece 1 To.

Forms
Microsoft Forms es una herramienta de encuestas en línea integrada en la suite de Microsoft Office 365. Ofrece una amplia gama de opciones para crear encuestas. Una vez creado, puede compartir encuestas por correo electrónico o redes sociales, y se puede solicitar a los usuarios que respondan con sus cuentas de Microsoft.

Stream
Microsoft Stream es un servicio de video empresarial basado en la nube diseñado para aumentar las comunicaciones y la colaboración entre equipos. Al igual que Slack, Microsoft Stream combina videoconferencias, mensajería en tiempo real y uso compartido de archivos en un solo lugar.

Sway
Hay varias maneras de crear una hermosa apariencia profesional. informe, presentación u otros documentos, pero Microsoft Sway es la forma más fácil, mejor y más eficiente de hacerlo. Sway es un entorno de trabajo integrado que facilita la creación y presentación de una colección de imágenes, documentos y otro contenido en una sola página web, perfecto para presentaciones. No necesita aprender nada nuevo para usar Sway. Es rápido y fácil, y funciona con sus archivos de Microsoft Office

 

Teams
•   Cuenta con espacio para colocar materiales de todo tipo y permitir al estudiante tener acceso a los mismos.
•   Brinda herramientas de trabajo para comunicarse con los estudiantes, crear y compartir contenido, de forma activa y productiva.
•   Provee espacios de comunicación y colaboración para todo el grupo, grupos pequeños y de forma individual.

Skype

Herramienta de intercomunicación para compartir experiencias y novedades. A nivel educativo permite trasladar el aula en tiempo real a espacios lejanos.
